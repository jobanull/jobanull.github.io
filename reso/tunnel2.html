Tunnel VPN (PPTP, L2TP, SSTP, OVPN)

Tunnel VPN adalah jalur terenkripsi antara perangkat client dan server, untuk melindungi data saat melewati jaringan publik. Tunnel VPN memungkinkan koneksi remote yang aman, site-to-site, atau akses jaringan internal dari lokasi eksternal.

1. PPTP (Point-to-Point Tunneling Protocol)

Protokol VPN layer 2 yang menggunakan GRE + MPPE untuk enkripsi. Mudah dikonfigurasi dan cepat, tapi **tidak aman** untuk data sensitif. Cocok hanya untuk lab atau testing.

CONTOH KASUS

  • PPTP

PPTP-RB1


/ip pool
add name=pptp_pool ranges=10.10.10.2
/ppp profile
add local-address=10.10.10.1 name=pptp_profile remote-address=pptp_pool
/interface pptp-server server
set enabled=yes
/ip address
add address=192.168.190.100/24 interface=ether1 network=192.168.190.0
add address=192.168.1.1/30 interface=ether2 network=192.168.1.0
/ip route
add distance=1 dst-address=192.168.2.0/30 gateway=10.10.10.2
/ppp secret
add name=pptp1 password=pptp1 profile=pptp_profile service=pptp
/system identity
set name=PPTP-R1

PPTP-RB2


/ip address
add address=192.168.190.101/24 interface=ether1 network=192.168.190.0
add address=192.168.2.1/30 interface=ether2 network=192.168.2.0
/ip dhcp-server network
add address=192.168.2.0/30 gateway=192.168.2.1
/ip route
add distance=1 dst-address=192.168.1.0/30 gateway=10.10.10.1
/system identity
set name=PPTP-R2

2. L2TP/IPsec

VPN layer 2 yang digabungkan dengan IPsec untuk keamanan. Mendukung multi-protocol transport dan autentikasi kuat. Lebih aman dari PPTP, tapi overhead enkripsi membuatnya sedikit lebih lambat.

CONTOH KASUS

  • L2TP

L2TP-RB1


/ip pool
add name=l2tp_pool ranges=10.10.10.3
/ppp profile
add local-address=10.10.10.1 name=l2tp_profile remote-address=l2tp_pool
/interface l2tp-server server
set default-profile=default enabled=yes ipsec-secret=rahasia use-ipsec=yes
/ip address
add address=192.168.190.104/24 interface=ether1 network=192.168.190.0
add address=192.168.1.1/30 interface=ether2 network=192.168.1.0
/ip route
add distance=1 dst-address=192.168.2.0/30 gateway=10.10.10.3
/ppp secret
add name=l2tp1 password=l2tp1 profile=l2tp_profile service=l2tp
/system identity
set name=L2TP-R1

L2TP-RB2


/interface l2tp-client
add connect-to=192.168.190.104 disabled=no ipsec-secret=rahasia name=\
    l2tp-out1 password=l2tp1 use-ipsec=yes user=l2tp1
/ip pool
add name=dhcp_pool0 ranges=192.168.2.2
/ip dhcp-server
add address-pool=dhcp_pool0 disabled=no interface=ether2 name=dhcp1
/ip address
add address=192.168.190.105/24 interface=ether1 network=192.168.190.0
add address=192.168.2.1/30 interface=ether2 network=192.168.2.0
/ip dhcp-client
add disabled=no interface=ether1
/ip dhcp-server network
add address=192.168.2.0/30 gateway=192.168.2.1
/ip route
add distance=1 dst-address=192.168.1.0/30 gateway=10.10.10.1
/system identity
set name=L2TP-R2

3. SSTP (Secure Socket Tunneling Protocol)

Tunnel VPN yang berjalan di atas HTTPS (TCP 443). Menggunakan SSL/TLS untuk enkripsi sehingga aman dan mudah melewati firewall atau NAT. Cocok untuk client-to-site. 


# Server SSTP
/interface sstp-server server
set enabled=yes certificate=server-cert verify-client-certificate=no

4. OpenVPN (OVPN)

VPN open-source yang fleksibel, berjalan di TCP atau UDP, menggunakan SSL/TLS untuk enkripsi kuat. Sangat aman, cross-platform, dan dapat melewati firewall/NAT. Cocok untuk penggunaan produksi. 


# Server OpenVPN
/interface ovpn-server server
set enabled=yes port=1194 mode=ip netmask=24 require-client-certificate=yes
/ppp secret
add name=user1 password=rahasia service=ovpn

2. L2TP/IPsec

VPN layer 2 yang digabungkan dengan IPsec untuk keamanan. Mendukung multi-protocol transport dan autentikasi kuat. Lebih aman dari PPTP, tapi overhead enkripsi membuatnya sedikit lebih lambat.

/system identity

add disabled=no interface=ether1 name=pppoe-out1 password=pppoe1 \
    service-name=PPPOE_Server user=pppoe1
/interface wireless security-profiles
set [ find default=yes ] supplicant-identity=MikroTik

add name=dhcp_pool0 ranges=192.168.2.2
/ip address
add address=192.168.190.109/24 interface=ether1 network=192.168.190.0

Tips

  • PPTP hanya untuk lab/testing, hindari di produksi.
  • L2TP/IPsec cocok jika butuh keamanan dan multi-platform.
  • SSTP cocok jika client berada di jaringan dengan firewall ketat.
  • OpenVPN direkomendasikan untuk produksi karena fleksibel dan aman.
