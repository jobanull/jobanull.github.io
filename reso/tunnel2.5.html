PPPoE (Point-to-Point Protocol over Ethernet)

PPPoE adalah protokol yang mengenkapsulasi PPP dalam frame Ethernet. Umumnya digunakan untuk koneksi broadband DSL atau antar-site point-to-point melalui Ethernet. Meskipun bukan VPN, PPPoE menyediakan autentikasi, enkapsulasi, dan session management.

  • Tujuan: Menghubungkan client ke server menggunakan autentikasi PPP (username/password) melalui Ethernet.
  • Use Case: ISP DSL, point-to-point antar kantor, manajemen user di layer 2.
  • Kelebihan: Autentikasi built-in, session management, sederhana untuk koneksi point-to-point.
  • Keterbatasan: Tidak menyediakan enkripsi; throughput terbatas oleh overhead PPPoE (~8 byte/header tambahan).

CONTOH KASUS

  • Menghubungkan dua router menggunakan PPPoE.
  • RB1 sebagai PPPoE Server, RB2 sebagai PPPoE Client.
  • Digunakan untuk menghubungkan network 192.168.1.0/30 dengan 192.168.2.0/30.

PPPoE-RB1 (Server)


/ip pool
add name=pppoe_pool ranges=10.10.10.4                          # Pool IP untuk PPPoE client (RB2 akan mendapat IP ini)

/ppp profile
add local-address=10.10.10.1 name=pppoe_profile \
    remote-address=pppoe_pool                                  # Profil PPP: RB1 = 10.10.10.1, client ambil dari pool

/interface pppoe-server server
add default-profile=pppoe_profile disabled=no interface=ether1 \
    service-name=PPPOE_Server                                  # Aktifkan PPPoE server di ether1

/ip address
add address=192.168.190.108/24 interface=ether1                # IP WAN RB1
add address=192.168.1.1/30 interface=ether2                    # IP LAN RB1

/ip route
add distance=1 dst-address=192.168.2.0/30 gateway=10.10.10.4   # Routing ke LAN RB2 via IP PPPoE client

/ppp secret
add name=pppoe1 password=pppoe1 profile=pppoe_profile \
    service=pppoe                                              # Username & password PPPoE untuk autentikasi client

/system identity
set name=PPPOE-R1                                              # Nama router

PPPoE-RB2 (Client)


/interface pppoe-client
add disabled=no interface=ether1 name=pppoe-out1 \
    user=pppoe1 password=pppoe1 service-name=PPPOE_Server      # PPPoE client terhubung ke server RB1

/interface wireless security-profiles
set [ find default=yes ] supplicant-identity=MikroTik          # Identitas supplicant default (opsional, untuk wireless)

/ip pool
add name=dhcp_pool0 ranges=192.168.2.2                         # Pool DHCP LAN RB2

/ip address
add address=192.168.190.109/24 interface=ether1                # IP WAN RB2
add address=192.168.2.1/30 interface=ether2                    # IP LAN RB2

/ip route
add distance=1 dst-address=192.168.1.0/30 gateway=10.10.10.1   # Routing ke LAN RB1 via IP PPPoE server

/system identity
set name=PPPOE-R2                                              # Nama router

Tips Praktis

  • PPPoE cocok untuk point-to-point atau ISP DSL, tetapi overhead kecil perlu diperhatikan jika banyak session.
  • Gunakan pool IP yang cukup untuk semua klien agar tidak terjadi konflik IP.
  • Pastikan autentikasi username/password benar dan interface sudah diatur sesuai topologi.
