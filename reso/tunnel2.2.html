OpenVPN (OVPN) - VPN

OpenVPN adalah protokol VPN open-source yang menggunakan SSL/TLS untuk enkripsi, cocok untuk remote-access dan site-to-site VPN. OpenVPN mendukung TCP dan UDP, serta enkripsi kuat untuk keamanan data.

  • Tujuan: Membuat koneksi VPN aman antar site atau untuk akses remote user ke jaringan kantor.
  • VPN: Data dienkripsi menggunakan SSL/TLS, termasuk autentikasi sertifikat dan username/password.
  • Use Case: Remote access VPN, site-to-site VPN, mengamankan koneksi publik ke private network.
  • Kelebihan: Enkripsi kuat, open-source, dapat berjalan di TCP/UDP, lintas platform (Windows, Linux, macOS, MikroTik).
  • Keterbatasan: Konfigurasi lebih kompleks dibanding L2TP/PPTP; butuh sertifikat dan manajemen CA.

Tips Praktis

  • Gunakan sertifikat CA sendiri untuk autentikasi klien, jangan hanya user/password.
  • OpenVPN di MikroTik berjalan di TCP, pastikan port 1194 terbuka di firewall/NAT.
  • Gunakan IP pool berbeda dari jaringan lokal agar routing lebih mudah.
